# نظام المستودعات والمبيعات (Inventory & Sales System)

هذا التطبيق هو نظام إدارة مستودعات ومبيعات متكامل مبني باستخدام فلاتر (Flutter)، ويدعم اللغتين العربية والإنجليزية.

## الميزات الأساسية
- **دعم اللغتين (Bilingual):** واجهة مستخدم تدعم اللغتين العربية (RTL) والإنجليزية (LTR) بضغطة زر.
- **الوضع الداكن والفاتح (Dark/Light Mode):** إمكانية تغيير مظهر التطبيق لراحة العين.
- **تصميم متجاوب (Responsive Layout):** التطبيق يعمل بكفاءة على الشاشات الصغيرة (الجوال) عبر قائمة جانبية منسدلة (Drawer) والشاشات الكبيرة (سطح المكتب/الويب) عبر قائمة جانبية ثابتة (Sidebar).
- **بطاقة الصنف:** لإدارة وتسجيل الأصناف المختلفة بالمستودع.
- **سند الإدخالات:** لإدارة استلام البضائع والمشتريات.
- **سند الصرف:** لإدارة صرف البضائع والمبيعات.
- **العملاء والموردين:** سجل للتعامل مع أطراف النظام من موردين وعملاء.
- **التقارير:** استعراض إحصائيات النظام ومراقبة الأداء.

## التقنيات المستخدمة
- **Flutter:** إطار العمل الأساسي (تدعم Web, iOS, Android, Desktop).
- **State Management:** الاعتماد على `ChangeNotifier` مدمج خفيف لإدارة الحالة.
- **Localization:** استخدام `flutter_localizations` للتعامل التلقائي مع اتجاه النصوص وتطبيقات RTL.

## طريقة التشغيل

1. تأكد من تثبيت Flutter وتحديثه إلى الإصدار 3.7.2 أو أعلى.
2. قم بتثبيت الحزم اللازمة:
   ```bash
   flutter pub get
   ```
3. تشغيل التطبيق على الويب أو الجهاز:
   ```bash
   flutter run -d chrome
   ```
أو
   ```bash
   flutter run
   ```
